C++提供了两种实现：

- 宿主式（hosted）：

  宿主的意思是设计上C++程序能运行在在操作系统存在情况下可能提供的实现。

- 独立式（freestanding）：

  独立的意思是设计上C++程序能运行**在没有操作系统的极端情况下**使用的实现。

  >用这些头文件以外的任何头文件，任何功能，都会导致你的代码没有办法最大限度的可移植。
