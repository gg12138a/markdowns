# Socket的分类

Socket提供了两种通信机制：

- 基于流的socket
  - 基于TCP协议：有序、可靠、双向字节流的通道
- 基于数据包的socket
  - 基于UDP协议：不需要建立和维护链接。对数据的长度有限制，但效率高。
