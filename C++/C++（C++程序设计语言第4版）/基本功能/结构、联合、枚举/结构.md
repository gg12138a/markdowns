# struct的内存布局

在struct的对象中，成员按照声明的顺序依次存放。



例如：
```C++
struct Readout{
  	char hour;
    int value;
    char seq;
};
```

![image-20220721141334623](%E7%BB%93%E6%9E%84.assets/image-20220721141334623.png)



一个struct对象的大小，不一定恰好等于它所有元素大小的累积之和。

因为很多机器，要求一些特定类型的对象，沿着系统结构设定的边界分配空间，以便机器能高效地处理这些对象。如整数往往沿着字的边界分配空间。
