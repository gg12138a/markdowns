# char* 与 char[]

```C++
#include <iostream>
#include <cstring>

int main() {
    char *ch_ptr = "hello";

    char ch_arr[] = "hello";

    using namespace std;

    cout << strlen(ch_ptr) << endl; // 5
    cout << strlen(ch_arr) << endl; // 5

    cout << sizeof(ch_ptr) << endl; // 8（为指针大小）
    cout << sizeof(ch_arr) << endl; // 6（个字符）

    return 0;
}
```



- 对char*，使用sizeof，得到的是指针的大小
- 对char[]，使用sizeof，得到的数组占用的大小（算上‘\0'）
